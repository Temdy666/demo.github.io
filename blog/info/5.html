UEditor富文本编辑器集成PrismJS代码高亮插件

Temdy 2021-08-28 02:58:11 【Java Code】 437人已围观

UEditor是由百度web前端研发部开发所见即所得富文本web编辑器，具有轻量、可定制、注重用户体验等特点。

PrismJS是一款轻量级、可扩展的语法高亮显示工具，在支持现代 Web 标准基础下增加了更多可选的风格插件。

UEditor富文本编辑器本身就已经集成了SyntaxHighlighter代码高亮插件，为什么还要集成PrismJS代码高亮插件呢？

答案是因为SyntaxHighlighter代码高亮插件显示太丑了，做为一个逼格高的站长是不会允许这种事情发生哒，所以只能找一款漂亮的来代替它，本站长选择的是PrismJS。

为什么选择 PrismJS ？

极致易用

引用 prism.css 和 prism.js，使用合适的 HTML5 标签（code.language-xxxx），搞定！

天生伶俐

语言的 CSS 类是可继承的，所以你只需定义一次就能应用到多个代码片段。

轻如鸿毛

代码压缩后只有 1.6KB。每添加一个语言平均增加 0.3-0.5KB，主题在 1KB 左右。

快如闪电

如果可能，支持通过 Web Workers 实现并行。

轻松扩展

定义新语言或扩展现有语法，或者新增功能都非常简单。

丰富样式

所有的样式通过 CSS 完成，并使用合理的类名如：.comment, .string, .property 等。


接下来教会大家如何在不修改ueditor编辑器的任何代码的情况下2个步骤的实现使用PrismJS。

1、在内容输出之前把SyntaxHighlighter代码高亮插件的代码替换成PrismJS代码高亮插件的代码

SyntaxHighlighter的格式是<pre class="brush:java;toolbar:false"></pre>

PrismJS的格式是<pre><code class="language-css"></code></pre>

后台代码实现（java语言）

/**
 * 转换Prism代码高亮插件内容的方法
 * @param content
 * @return
 */
public static String toPrism(String content) {
	Document doc = null;
	try {
		doc = Jsoup.parse(content);
		Elements elements = doc.select("pre");
		String html = null;
		String css = null;
		String language = null;
		for (Element el : elements) {
			html = el.html();
			Element code = doc.createElement("code");
			css = el.attr("class");
			language = css.substring(6,css.indexOf(";"));
			code.addClass("language-" + language);
			code.html(html);
			el.html("");
			el.removeClass(css);
			code.appendTo(el);
		}
		//内容转换成Document对象后会产生多余的html标签，最后要处理掉
		content = doc.html()
				.replace("<html>","")
				.replace("<head></head>","")
				.replace("<body>","")
				.replace("</body>","")
				.replace("</html>","");
	} catch (Exception e) {
		e.printStackTrace();
	}
	return content;
}

前端页面实现

/**
 * 转换Prism代码高亮插件内容的方法
 * @param content
 * @return
 */
function toPrism(content){
	// 创建一个 div 来放置获取到的内容，这样就可以把 content 字符串内容转换成 dom
	// 方便我们对这个 dom 进行操作（只是创建 div，并没有插入到文档，他只存在于内存中）
	var parent = document.createElement('div');
	parent.innerHTML = content;
	
	// 找到 parent 中的所有 pre 标签，并遍历
	var $pres = $(parent).find('pre');
	$pres.each(function(index, el) {
		var $el = $(el),
			html = $el.html(),
			code = document.createElement('code'),
			elClass = $el.attr('class'),
			// 获取 pre 标签中，class 属性中包含的本段代码所有的编程语言
			language = elClass.substring(6, elClass.indexOf(';'));
		$el.removeAttr("class");
		code.className = 'language-' + language;
		code.innerHTML = html;
		$el.html(code);
	});
	return parent.innerHTML;
}

// 最后，调用 Prism 的方法来高亮代码
Prism.highlightAll();

<script type="text/javascript" src="prism.js"></script>

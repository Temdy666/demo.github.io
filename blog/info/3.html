Springboot程序HelloWord的404错误问题

Temdy 2021-08-25 10:22:11 【Spring-boot】 417人已围观

问题的本质在于区分@Controller和@RestController注解的功能使用，前者用于解析视图跳转页面，后者是直接响应文本内容，一般@Controller+@ResponseBody配合灵活使用多。

解决的这个问题的方案有以下两种：

1、使用@Controller注解的话，方法里面return的字符串是解析视图文件名路径，所以系统就直接报404找不到文件的错误，只要在方法上加上@ResponseBody注解来告诉系统响应文本内容就解决了。

@Controller
public class HelloWordController {
    @RequestMapping("/")
    @ResponseBody
    public String sayHello(){
        return "helloword";
    }
}

 

2、直接使用@RestController代替@Controller注解就直接解决，@RestController等价于@Controller和@ResponseBody。

@RestController
public class HelloWordController {
    @RequestMapping("/")
    public String sayHello(){
        return "helloword";
    }
}
